Test: Same-Domain Universal Link /finish

Ziel des Tests: Prüfen, ob ein Universal Link auf der gleichen Domain (kein Subdomain-Wechsel) aus einer iOS-WebView heraus die App auslöst.
Ablauf:
  1. User klickt DeepLink: passkeyguard://open?origin=...&returnURL=...
  2. App öffnet WebView mit https://www.appsprint.de/samedomain-finish.html
  3. In der WebView wird der Universal Link https://www.appsprint.de/finish ausgelöst (gleiche Domain!)
  4. Test: Fängt iOS den Link ab (App öffnet) oder lädt stattdessen die /finish-Seite in der WebView?
